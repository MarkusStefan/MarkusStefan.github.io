Notes on Typography

2025-08-22

Typography is the UI you read. Some gentle defaults:

  • Comfortable line-height

  • Avoid long measure

  • Honest contrast (no gray-on-gray)

Type rhythm

Subtle spacing makes the page breathe. Headlines should set a pace, not shout.
